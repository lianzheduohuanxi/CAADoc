## ⚠️ 仍需关注的格式问题

### 1. HTML表格残留

**现状**: V4 转换器跳过了表格结构，内容以文本形式输出

**示例**:
```markdown
![image](../../assets/images/atarget.gif)
|[]This macro shows you how to create an 
Analysis document for a generative structural analysis...
```

**影响**: 可读性略有下降，但内容完整性得到保证

**可选方案**:
1. 手动整理关键表格
2. 开发更智能的表格转换器

### 2. 代码块格式

**现状**: VBScript 代码块保留良好，但有断行问题

**示例**:
```vbscript
```vbscript
Set 
```
```vbscript
documents1 = CATIA.Documents
```
```

**原因**: HTML 中的 `<pre>` 标签格式化问题

### 3. 图片路径

**现状**: `./assets/images/` 相对路径保留

**影响**: 在 Markdown 渲染器中可能无法正确显示

**可选方案**:
1. 下载图片到知识库目录
2. 使用绝对 URL
