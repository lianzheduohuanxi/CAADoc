# CATIA CAA V5 知识库蒸馏计划

## 📋 项目概述

本项目是 CATIA V5 CAA (Component Application Architecture) 官方知识库，需要将其蒸馏为 pi agent 可用的格式。

### 原始数据规模
- **源码示例**: 67个 `.edu` 模块（约2GB+源码）
- **在线文档**: 321个主题目录
- **文档类型**: HTML使用案例 + 技术文章 + API参考

## 🔄 转换策略

### 1. HTML文档 → Markdown

```bash
# 转换脚本伪代码
For each .htm file in Doc/online/:
    1. 提取标题和内容
    2. 解析代码示例块
    3. 提取关键API引用
    4. 生成带有frontmatter的Markdown
    5. 输出到 knowledge/markdown/{category}/
```

**转换规范**:
- Frontmatter 包含: title, category, tags, source_file
- 代码块保留原始格式
- 提取 `CATI*`, `CAA*` 接口名为标签

### 2. C++源码 → 结构化知识

```cpp
// 提取模式
提取内容:
1. 类名、接口名
2. 方法签名
3. QI (QueryInterface) 路径
4. #include 依赖
5. 示例代码片段

输出格式: JSON知识条目
```

### 3. API参考 → 可查询数据库

```json
{
  "interface": "CATIProduct",
  "module": "CAAProductStructure",
  "methods": [...],
  "related_interfaces": [...],
  "use_cases": [...],
  "code_examples": [...]
}
```
