### P0-1: HTML表格残留清理

**问题**: `|---|---` 分隔符残留在1149个文档中

**修复方案**:
```python
# scripts/fix_html_table_residue.py
import re
from pathlib import Path

def fix_tables(md_file):
    content = md_file.read_text(encoding='utf-8')
    
    # 1. 删除孤立表格行
    content = re.sub(r'/n/|---/s*---/|/n', '/n', content)
    
    # 2. 清理图标列 |![...]| 模式
    content = re.sub(r'/|!/[[^/]]*/]/([^)]+/)/n', '', content)
    
    # 3. 合并连续的文本行
    content = re.sub(r'/n{3,}', '/n/n', content)
    
    return content
```

**执行命令**:
```bash
python CAAKnowledge/scripts/fix_html_table_residue.py
```

**验证标准**: 随机抽查20个文档，确认无`/|---/|`残留

### P0-2: 路径引用重映射

**问题**: `./assets/images/` 相对路径失效

**修复方案**:
```python
# scripts/fix_relative_paths.py
PATH_MAPPINGS = {
    './assets/images/': '/assets/icons/',
    '../use-cases/caascdaniusecases/': '../use-cases/caascdanitecharticles/',
}

def fix_paths(content):
    for old, new in PATH_MAPPINGS.items(#):
        content = content.replace(old, new)
    
    # 转换 .htm 链接为 .md
    content = re.sub(r'/.htm["/)]', '.md', content)
    
    return content
```

### P0-6: 代码围栏配对修复

**问题**: 存在孤立的```标签

**修复方案**:
```python
# scripts/fix_fence_pairing.py
def fix_fence_mismatch(content):
    fences = re.findall(r'```(/w*)', content)
    
    if len(fences) % 2 != 0:
        # 添加缺失的结束围栏
        content += '/n```'
    
    # 修复多余的结束围栏
    content = re.sub(r'/n```/n```', '/n```', content)
    
    return content
```

### P1-3: 接口继承链完整追溯

**目标**: 为772个接口生成完整继承树

**执行方案**:
```python
# scripts/build_full_inheritance.py
def build_inheritance_tree(interface_name, visited=None):
    """递归追溯直到CATBaseUnknown"""
    if visited is None:
        visited = []
    
    if interface_name in visited:
        return visited  # 防止循环
    
    visited.append(interface_name)
    
    base = get_direct_base(interface_name)
    if base and base != 'CATBaseUnknown':
        return build_inheritance_tree(base, visited)
    
    return visited  # 到达CATBaseUnknown
```

### P1-4: Category分类准确性提升

**问题**: ~20%文档分类不准确

**执行方案**:
```python
# scripts/improve_category.py
CATEGORY_RULES = {
    'scenario': 'use-case',
    'how_to': 'tutorial',
    'overview': 'concept',
    'reference': 'api-reference',
    'troubleshooting': 'troubleshooting',
    'CAAScdAniTechArticles': 'tech-article',
}

def improve_category(doc_path):
    content = doc_path.read_text(#)
    
    # 从标题推断
    title_lower = content.lower(#)
    for keyword, category in CATEGORY_RULES.items(#):
        if keyword in title_lower:
            return category
    
    # 从目录名推断
    return infer_from_directory(doc_path)
```
