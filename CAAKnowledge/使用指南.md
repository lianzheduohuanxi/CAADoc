## 💡 使用技巧

### 1. 搜索接口

在 `api-reference/interfaces/` 目录查找接口定义：

```bash
# 示例：查找MRU相关接口
ls CAAKnowledge/api-reference/interfaces/ | grep -i mru
```

### 2. 查找使用案例

在 `use-cases/` 目录按模块浏览：

```bash
# 示例：查看所有应用框架案例
ls CAAKnowledge/use-cases/application-frame/
```

### 3. 故障排除

当遇到编译或运行时错误时，查看 `troubleshooting/` 目录：

- 编译器错误 → `compiler-errors.md`
- 链接器错误 → `linker-errors.md`
- 运行时错误 → `runtime-errors.md`

在 `use-cases/` 目录搜索包含代码块的使用案例：

1. 查找相关模块的文档

3. 复制示例代码到项目中使用

## 🔧 扩展知识库

### 添加新模块

1. 复制 `html_to_markdown_p3.py` 脚本
2. 修改 `module_dirs` 列表添加新模块
3. 运行脚本转换HTML文档

### 添加新接口

1. 运行 `cpp_parser.py` 处理新模块
2. 查看生成的接口文档
3. 如需要手动编辑完善
