# CAAEPstINFModelEvents

> Data extension of the CATProduct feature objects, implementing the CATIModelEvents interface to enable the revisualization of the objects after a modification. Illustrates programming the sending of events on the modification of an object by implementing the CATIModelEvents interface of the Visualization framework. This class derives from the CATExtIModelEvents adapter which implements all the interface services. Inheritance: CATExtIModelEvents (Visualization Framework) CATBaseUnknown (System Framework)
