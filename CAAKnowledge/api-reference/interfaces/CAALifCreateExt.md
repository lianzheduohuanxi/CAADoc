# CAALifCreateExt

> Required to call the user function code at run-time. CAALifCreateExt is a code extension of CAALifDicoLibrary whose purpose is to declare a user function. This extension should implement CATICreateInstance to be able to create dynamically a CAALifDicoLibrary object.
