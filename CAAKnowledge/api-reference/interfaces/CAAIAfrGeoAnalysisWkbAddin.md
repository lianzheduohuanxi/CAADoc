# CAAIAfrGeoAnalysisWkbAddin

> Interface that the addins of the CAAAfrGeoAnalysisWkb workbench must implement. An addin enables to add new toolbars into a workbench. The workbench returns this interface name in its GetAddinInterface method. This interface must derive from CATIWorkbenchAddin.
