# CAASysCollectionEmptyNotif

> Notification class. An instance of this class is created and sent by the the implementation of CAAISysCollection interface whenever it becomes empty. In the same way, a CAASysCollectionFilledNotif is sent when the collection becomes not empty. Inheritance: CAASysCollectionNotification ( Local Framework) CATNotification  ( System Framework) CATBaseUnknown (System Framework).
