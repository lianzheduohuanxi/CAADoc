## Infrastructure
 
 
## []Adding a Macro as a Command in a Toolbar
 
 

---

You can easily customize your toolbars by adding icons to trigger your
favorite macros. This reduces the macro access time for the macros you often
run.

 
 |![](images/atarget.gif)
 

This task explains how to add a macro as a command in a toolbar.
 
 
 
 |![](images/ascenari.gif)
 

 
- Select the **Tools->Macro->Macros... **command to display
 the **Macros **dialog box, and make sure that the directory path is added in '**Current 
		Macro library or documents**' section. 
		 All macros under this directory are listed in '**Available macros**' section.
		 Then click '**Close**'

 ![](images/CAAInfAdd1.jpg)
 
 
 
 

 
- Select **Tools->Customize** , then the '**Commands**' tab,
 and Macros. Select the '**Macros**' under '**Categories**'
 ![](images/CAAInfAdd2.jpg) 
 
 
 
 
 
 

 
- In Customize window select '**Show Properties**'
 The default icon ![](images/CAAInfDefaultIcon.jpg) is used to show Icon.

 ![](images/CAAInfAdd3.jpg)
	
 
	
 
 
 
 
 
 

 
- To select another icon than the default one,click on 
 ![](images/CAAInfIicon1.jpg)
 
 
 
 

 
- Select the appropriate icon from '**Icon Browser**'

 ![](images/CAAInfAdd3.jpg)
