CAA V5 Encyclopedia |  New C++ Authorized APIs in CAA V5R19 SP2  

* * *

The following are the new CAA V5R19 SP2 C++ Authorized APIs, compared with CAA V5R19 at SP1 level.

  * AdvancedTopologicalOpe framework 
    * Enumeration CATTopologicalFillType
  * CATIAApplicationFrame framework 
    * Class CATCafSearchInformationAdapter
    * Interface CATISearchInformation
  * CATPDMBase framework 
    * Global Function SaveToPDM
  * ObjectModelerNavigator framework 
    * Enumeration CATNavigNodeState
  * TopologicalOperators framework 
    * Enumeration CATCGMDynTransformationReportMode
  * VPMInterfaces framework 
    * Macro or #define EFFVALSIZE
    * Interface CATICfgEffVal
