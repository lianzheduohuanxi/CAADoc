CAA V5 Encyclopedia |  New C++ Authorized APIs in CAA V5R8 SP1

* * *

The following are the new CAA V5R8 SP1 C++ Authorized APIs, compared with CAA V5R8 at GA level.

  * AdvancedMathematics framework
    * CATMathFoundationX
    * CATMathFoundationXY
    * CATMathFunctionGeneral
  * ApplicationFrame framework
    * AddAccessChild
    * SetAddinMenu
  * CATIAApplicationFrame framework
    * CATExtISelectShow
    * CATICafTreeSettingAtt
    * GetCATCafTreeSettingCtrl
  * CATTPSInterfaces framework
    * CATITPSAxisStraightness
    * CATITPSView
    * CATITPSViewList
  * Dialog framework
    * CATDlgDiaFINISHNotification
    * CATDlgDiaNEXTNotification
    * CATDlgDiaPREVIOUSNotification
  * ENOVInterfaces framework
    * ENOVIBOActionManager
    * ENOVIBOEcoManager
    * ENOVICWAffectedObjects
    * ENOVICWConfiguredViews
    * ENOVICWLinks
    * ENOVICWState
    * GetBOActManager
    * GetBOManager
  * GSMInterfaces framework
    * CATGSMCornerType
    * CATGSMExtrapolPropagationMode
    * CATGSMLinearDraftSweepComp
    * CATGSMLinearSweepLengthType
    * CATIGSMGridSet
  * ManufacturingInterfaces framework
    * CATIMfgDesignHoleLimits
  * ObjectModelerSDM framework
    * CreateModelFromSchema
    * DeleteModel
    * GetContainerFromModel
    * GetModelFromContainer
    * GetModelFromFile
    * SaveFileFromModel
  * ObjectSpecsModeler framework
    * CATIBuildProvider
  * PSNInteroperability framework
    * CATCommitVPM
    * CATSaveInVPMFromCATIA
  * VPMInterfaces framework
    * CATICfgSpecification
  * Visualization framework
    * CATISelectShow
  * VisualizationBase framework (created in V5R8 in Visualization. This framework was split into Visualization and VisualizationBase in V5R9)
    * CAT3DArrowGP
