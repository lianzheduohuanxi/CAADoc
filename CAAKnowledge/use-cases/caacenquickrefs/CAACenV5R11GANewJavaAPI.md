CAA V5 Encyclopedia |  New Java Authorized APIs in CAA V5R11 GA

* * *

The following are the new CAA V5R11 GA Java Authorized APIs, compared with CAA V5R10 at GA level.

  * The new catjdialog package com/dassault_systemes/catjdialog
  * com/dassault_systemes/catweb/base/catlet Package
    * JCATletAbstract
    * JCATletAbstractBeanInfo
  * com/dassault_systemes/catweb/base/catlet/model Package
    * ModelAbstract
  * com/dassault_systemes/enovdesktopappli/vdk0appli/interfaces Package
    * ENOVIItemInstanceModel
    * ENOVIPECATlet
  * com/dassault_systemes/vpmdesktop/vdk0base/infra Package
    * AbstractENOVAttribute
    * AbstractENOVObject
    * AbstractENOVType
  * com/dassault_systemes/vpmdesktop/vdk0infra/interfaces Package
    * ENOVISearchResult
  * com/dassault_systemes/vpmdesktop/vdk0interfaces/infra Package
    * ENOVGetFactories
    * ENOVTypeMask
  * com/dassault_systemes/vpmdesktop/vdk0interfaces/interfaces Package
    * ENOVIAttribute
    * ENOVIObject
    * ENOVIObjectFactory
    * ENOVIObjectList
    * ENOVIType
    * ENOVITypeFactory
