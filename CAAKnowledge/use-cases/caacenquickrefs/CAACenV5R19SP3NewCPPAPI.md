CAA V5 Encyclopedia |  New C++ Authorized APIs in CAA V5R19 SP3  
---|---  
  
* * *

The following are the new CAA V5R19 SP3 C++ Authorized APIs, compared with CAA V5R19 at SP2 level.

  * AdvancedTopologicalOpe framework 
    * Global Function CATCreateTopologicalFillLight
  * CATTPSInterfaces framework 
    * Interface CATITPSDimVisu
  * Drafting2DLInterfaces framework 
    * Enumeration CAT2DLClippingProfileMode
    * Interface CATI2DLayoutClipping
  * GeometricObjects framework 
    * Macro or #define CATSubdivGridSurfaceType
  * MecModInterfaces framework 
    * Class CATLISTV(CATIBRepAccess_var)
  * MechanicalModeler framework 
    * Class CATMmrImportComparator
  * System framework 
    * Macro or #define CATMaxPathSize
    * Interface CATISysFileAccessStatisticsSettingAtt
