Technical Article

* * *

Abstract This article presents by frameworks the detail of CAA Java resources modified in V5R10 and how to modify your code accordingly. Each modification is listed for a resource (file, class, method) with a classification. They will generally be detected by a recompilation of the code. Most of the time, signature modifications are obvious. Check in the framework detail how to replace deleted entities.
