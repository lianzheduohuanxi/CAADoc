## 🔴 严重问题 (必须立即修复)

### 问题 1: source_file 扩展名错误 (约 100 个文件)

**严重程度**: 严重  
**影响范围**: ~100 个文档

**问题描述**: `source_file` 字段指向 `.md` 文件而非原始 `.htm` 文件。

**错误示例**:
```yaml
# 错误
source_file: "Doc/online/CAAScdAniUseCases/CAAAniMeshBasicSurf.md"

# 正确
source_file: "Doc/online/CAAScdAniUseCases/CAAAniMeshBasicSurf.htm"
```

**根因**: 修复脚本错误地将所有 `.htm` 链接转换为 `.md`，包括 `source_file` 字段。

**影响**: 
- 无法追溯原始 HTML 源文件
- 文档溯源能力丧失
- 数据完整性损坏

**修复方案**:
```python
# 修复脚本
for md_file in use_cases_dir.rglob('*.md'):
    with open(md_file, 'r') as f:
        content = f.read()
    
    # 只修复 source_file 中的路径
    content = re.sub(
        r'source_file:\s*"([^"]*)\.md"',
        lambda m: f'source_file: "{m.group(1)}.htm"',
        content
    )
    
    with open(md_file, 'w') as f:
        f.write(content)
```

### 问题 2: HTML 表格残留 (约 1149 个文件)

**严重程度**: 严重  
**影响范围**: ~1149 个文档  
**残留数量**: 368+ 处

**问题描述**: HTML 表格的分隔符 `|---|---` 被保留在 Markdown 中，形成格式错误的表格。

**错误示例**:
```markdown
|---  
| This macro shows... |
|---  
| Another step... |
```

**根因**: HTML 转 Markdown 时，未正确处理两列表格（一列图标，一列内容）。

**修复方案**:
1. 识别并删除无意义的单列空表格行
2. 将 `|---|---` 转换为正确的 Markdown 表格格式
3. 或将表格转换为列表格式

### 问题 3: 无效代码围栏配对 (约 100+ 个文件)

**严重程度**: 严重  
**影响范围**: 100+ 个文档

**问题描述**: 存在孤立的 ` ```vbscript ` 开始标签或结束标签，导致代码块无法正确渲染。

**错误示例**:
```markdown
# 错误: 标签配对错误
```vbscript
        If (Not CATIA.FileSystem.FolderExists(sDocPath)) Then
```vbscript
          Err.Raise 9999,,"No Doc Path Defined"
        End If
```vbscript

# 正确
```vbscript
If (Not CATIA.FileSystem.FolderExists(sDocPath)) Then
  Err.Raise 9999,,"No Doc Path Defined"
End If
```
```

**根因**: VBScript 代码块检测逻辑过于复杂，导致标签插入位置错误。

**当前说明**:
```markdown
## 说明

该接口没有在当前头文件中声明方法。继承自基类的潜在方法请查阅基类文档。
```

**问题**: 说明过于笼统，未说明实际用途。

### fix_source_file_ext.py
```python
#!/usr/bin/env python3
"""修复 source_file 扩展名，将 .md 改回 .htm"""
import re
from pathlib import Path

def fix_source_file_ext():
    use_case_dir = Path('CAAKnowledge/use-cases')
    fixed = 0
    for md_file in use_case_dir.rglob('*.md'):
        with open(md_file, 'r', encoding='utf-8') as f:
            content = f.read()
        
        new_content = re.sub(
            r'source_file:\s*"([^"]*)\.md"',
            lambda m: f'source_file: "{m.group(1)}.htm"',
            content
        )
        
        if new_content != content:
            with open(md_file, 'w', encoding='utf-8') as f:
                f.write(new_content)
            fixed += 1
    
    print(f"修复了 {fixed} 个文件的 source_file 扩展名")

if __name__ == '__main__':
    fix_source_file_ext()
```
