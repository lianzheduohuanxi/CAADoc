GMModelInterfaces Global Function CATCGMCreateBodyFromLengthOnWire


CATCGMCreateBodyFromLengthOnWire( iFactory,
iTopData,
iSamplingMeasure,
iMeasureType,
iWire,
iReferenceVertex,
iReverseSamplingDirection= FALSE,
iRelativeTolerance= 0.1)
Creates a CATICGMBodyFromLengthOnWire operator.
Parameters:
iFactory
The factory of the geometry.
iTopData
The pointer to the data defining the software configuration and the journal. If the journal inside iData is NULL, it is not filled.
iSamplingMeasure
Measure interval (arc lengths spacings or integrated turning angles) between the sample points on the wire. The wire will be sampled at iSamplingMeasure increments starting from the ReferenceVertex in the direction of wire orientation.
iMeasureType
Sampling measure type. Sets the operator to either use ArcLength or TurningAngle as the integral measure.
iWire
The wire domain to be sampled.
iReferenceVertex
The reference vertex where the sampling will start.
iReverseSamplingDirection
Reverse the sampling direction (with respect to the wire orientation)
iRelativeTolerance
Specifies the relative tolerance, a multiplicative factor of factory length or factory angular resolution used by this operator internally in the integration scheme to determine sample point positions on the wire. Lower tolerance values may result in a higher precision of the computation at the expense of a slower performance.
Returns:
[out, IUnknown#Release] The pointer to the created operator. To be released with the Release method after use.
