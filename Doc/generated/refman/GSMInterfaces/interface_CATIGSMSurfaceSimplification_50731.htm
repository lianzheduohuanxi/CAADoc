interface CATIGSMSurfaceSimplification

Interface to manipulate Surface Simplification feature.
Role: Allows you to access data of Surface Simplification feature. Note: requires GSO license.

Enumerated Types


o BoundaryConstraintLevel 
enum BoundaryConstraintLevel {
  UseG0InputKeepG1,
  UseG0InputUseG1Input
}
The boundary constraint level ( UseG0InputKeepG1, UseG0InputUseG1Input).
Parameters:
UseG0InputKeepG1
Mode to use only the distance deviation input.
UseG0InputUseG1Input
Mode to use both the distance deviation and the tangency deviation inputs.
