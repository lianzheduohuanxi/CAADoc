Editing Framework Identity Card

This task shows you how to edit the identity card of a framework, and how to update the prerequisites of your workspace accordingly.
Before you begin:

Open a workspace containing at least a framework.

  1. In the 3DS Workspace Explorer, right click the name of the framework for which you edit the identity card, and click Edit Id Card.

    The Identity Card Editor dialog box opens. See mkICE to use this editor.

  2. Edit and save this file.

  3. In the 3DS Workspace Explorer, right click the workspace name, and click Prerequisites.

    The 3DS - Manage prerequisites dialog box opens.

  4. Click OK.

    The workspace prerequisites are updated according to the identity card changes.
