3DS Workspace Finder

The 3DS Workspace Finder view shows all the workspace source files.

Before you begin:

Open the appropriate workspace. Note that the workspace must be built.

  1. Either:

    • On the View menu, point to 3DS Views, and click 3DS Workspace Finder.
    • Right click the root node of the 3DS Workspace Explorer, and click Search File.

    The 3DS Workspace Finder view opens.

  2. Click in front of the framework for which you want to display the files.

This view lists the names of the files of all the source files included in the workspace, along with their and their pathnames starting just below the workspace root folder. It includes a file filter, and shows the number of files listed with respect to the total workspace file count, such as 682/682 above.

The file filtering pattern can apply to the whole displayed pathnames, or to the file names only. The filter can be case sensitive.

Command Filtering Pattern
Full Path Check to apply the filtering pattern to the whole displayed file pathnames.
File Name Check to apply the filtering pattern to the file names only.
Case Sensitive Check to apply a case sensitive filter. Otherwise, the filter is case insensitive.

You can use the file filter as follows:

  • You can filter this list using a filtering pattern:

    1. In Filter, type a filtering pattern, for example *.h. The '*' character is the wild card character.

    2. Click Filter or press Enter.

      Only the files matching the filtering pattern are displayed.

      All the .h files of the workspace are listed. They are 38 files out of 100.

  • To apply the filtering pattern to the whole pathnames:

    1. In Filter, type a filtering pattern, for example CAADialog.

    2. Check Full Path to apply the filtering pattern to the whole pathnames.

    3. Click Filter or press Enter.

      All the files matching the CAADialog pattern in their whole pathnames are displayed, that is, in this case, all the files in the workspace. Note that the wild card character is not required.

  • To apply the filtering pattern to the file names only:

    1. In Filter, type a filtering pattern, for example CAADialog.

    2. Check File Name to apply the filtering pattern to the file names only.

    3. Click Filter or press Enter.

      All the files matching the CAADialog pattern in their file names only are displayed, that is, in this case, two files out of 100.
